# Git Commit & Authorship Guidelines
We utilize a multi-agent workflow powered by various open-source models. Because code is continuously generated, modified, and reviewed by multiple different agents, attempting to track which specific model wrote which line of code is inaccurate and pollutes our git history.

Therefore, you MUST NOT include any AI attribution, footers, or watermarks in your commit messages.
- OMIT all "Co-authored-by:" tags.
- OMIT all "Generated by" or tool-specific signatures.
- Output ONLY the standard commit subject and body.
